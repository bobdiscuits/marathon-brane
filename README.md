# The Marathon Brane

A community research vault for the lore of **Marathon** — Bungie's 1994 original, *Marathon 2: Durandal* (1995), *Marathon Infinity* (1996), and **Marathon (2026)**, Seasons 1–2.

Built as a **twin**: a word-for-word **Mirror** of the primary source text (every terminal, codex entry, and intercept — frozen, verbatim, headed per-terminal and per-entry), and a **Brane** of analysis hubs — characters, factions, places, concepts, threads — where **every factual claim is footnoted to that source**. Where the games stay silent, the pages say so plainly rather than guessing. Rich frontmatter on every hub (era, status, affiliation, scope, source counts) drives the type landings and the graph.

- **Browse:** start at the site's front door and follow the threads — the graph is the table of contents.
- **Take it with you:** `console.html` — the entire vault (all pages, search, and the 2,137-edge graph) in one offline HTML file.
- **Method:** the Twin-Vault Method — 1:1 capture → verify → segment → footnoted analysis under the "1:1 Guard" → machine-checked citation integrity (0 invalid citations vault-wide).
