# UESC Intercept

\=================================================================== 

RUNNERS' GUIDE                                      v.1.001.07.tciv

SCAVENGING: UESC INTERCEPT

\=================================================================== 

  

OPPORTUNITY: UESC INTERCEPT 

THREAT: UESC Security Forces, Rival RUNNERS 

RISK: Cross-ZONE navigation on a clock 

REWARD: UESC Crisis Supplies 

  

OVERVIEW: The UESC uses coded messaging to signal for emergency supply drops in contested ZONES. Because of the way they've hooked into old colony systems to improve their local comms, RUNNERS have figured out a way to mimic a UESC distress signal. Using colony access points to send a coded message that triggers a UESC crisis drop, a RUNNER or CREW can intercept the UESC response if they can reach the drop point without too many distractions [read: rivals RUNNERS or any other threat or opportunity that may pop up]. 

  

     • LOCATE a REGION with active colony access points 

     • LOCATE main access point 

     • NOTE numbered code on access point 

     • FIND corresponding access point 

     • TRIGGER mimic signal 

     • REACH drop point before UESC realize 

     • DAMAGE Crisis container under dropship 

     • LOOT Crisis container 

          • And survive UESC compliment 

     • SUCCESS! 

  

REMARK: Better to lay in wait than show up late.
